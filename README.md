
# CI/CD Pipeline with GitHub Actions

This project implements a complete CI/CD pipeline using GitHub Actions. It integrates industry-standard tools for security, code quality, containerization, and deployment to a Kubernetes cluster.




## Tech Stack
- GitHub Actions → CI/CD automation
- Docker → Containerization
- Kubernetes (K8s) → Deployment
- Trivy → File system & image vulnerability scanning
- Gitleaks → Hardcoded secrets scanning
- SonarQube → Code quality & static analysis
- Cloud Provider → AWS
